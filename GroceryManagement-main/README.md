🍃 Grocery Management System (Full Stack)

A full-stack Grocery Management System built using Spring Boot, React.js, and MySQL.
It includes user authentication, admin panel, product management, and order handling features.

🚀 Tech Stack
Backend: Java, Spring Boot, Spring Data JPA

Frontend: React.js

Database: MySQL

API: RESTful APIs

Tools: Maven, Git, GitHub

✨ Features

👤 User Side

User Registration & Login

View available grocery items

Place orders

🛠 Admin Panel

Add / Update / Delete products

Manage product inventory

View and manage orders

🧠 API Endpoints

POST /users/register → Register user

POST /users/login → Login user

GET /items → Get all products

POST /items → Add product (Admin only)

⚙️ How to Run

Backend

mvn spring-boot:run

Frontend

cd grocery-frontend

npm install

npm start
